# 更新日志

本项目按语义化版本记录用户可见的变化。

## 0.4.1 - 2026-07-11

### 修复

- 修复源码控制中心可能启动无触控能力的旧版 Windows 主机，导致手机触摸无响应的问题；
- 控制中心现在校验主机和设备启动器的文件版本，并优先使用与当前版本一致的构建产物；
- 检测到旧主机时只重启主机进程，保留现有扩展显示 Owner 和 Software Device，避免重复创建设备；
- Android 触摸视图现在始终接管完整手势，不再因发送线程瞬时未就绪而丢失后续移动或抬起事件。

### 改进

- 增加触摸握手能力、输入就绪状态和手势发送结果的结构化诊断日志；
- 在真实 Android 设备、1920 x 1080 FifScreen 扩展屏上验证 `WM_POINTER` 和鼠标兼容消息。

## 0.4.0 - 2026-07-11

### 新增

- Android 扩展屏支持原生单指和多指触控；
- 支持点击、拖动、长按以及由 Windows 应用处理的双指缩放和滚动手势；
- Windows 主机使用系统 `InjectTouchInput` 将触点注入 FifScreen 扩展显示器；
- 新增触控协议、坐标映射、单指、多指和断线取消测试。

### 改进

- Android 使用独立低延迟输入发送线程，并合并连续移动帧，避免触控积压；
- 握手增加触控能力协商，新旧 Android 和 Windows 组件可以兼容混用；
- 控制连接断开时自动取消所有活动触点，避免残留按下状态。

## 0.3.2 - 2026-07-10

### 修复

- 修复 GitHub Release 静态清单以二进制响应返回时，Windows PowerShell 5.1 无法解析 JSON、错误提示连接失败的问题；
- 静态清单现在先按 UTF-8 显式解码，再进行字段和 SHA-256 校验；
- 更新检查显式启用 TLS 1.2，并关闭 PowerShell 下载进度渲染以减少网络请求阻塞。

## 0.3.1 - 2026-07-10

### 新增

- Windows 控制中心启动时检查管理员权限；
- 非管理员启动时提示“请以管理员身份运行”，确认后立即退出；
- GitHub Release API 不可用时回退到最新 Release 的静态更新清单资产。

### 修复

- 修复网络正常但 GitHub API 间歇性超时导致手动检查立即报错的问题；
- 手动检查更新增加三次重试，只有 API 和静态清单都失败才显示网络提示；
- 后台检查继续使用静默退避重试，不打扰用户。

## 0.3.0 - 2026-07-10

### 新增

- Windows 控制中心启动时自动检查 GitHub Releases 最新版本；
- 更新提示显示版本号、发布时间和完整更新日志；
- 更新安装包自动下载、SHA-256 校验、运行时关闭和提权启动；
- GitHub 连接失败时后台静默退避重试；
- “关于”菜单、“检查更新”、“使用教程”和产品信息；
- 中文 Android 开发者模式、USB 调试和扩展屏使用教程；
- 可重复执行的 GitHub Release 发布脚本。

### 改进

- Windows 控制端按钮、状态和主要日志全面中文化；
- 安装器快捷方式、开发驱动警告和维护提示中文化；
- 安装包默认写入官方 GitHub Releases API 更新地址；
- README 改为完整中文项目说明。

## 0.2.1 - 2026-07-10

### 新增

- Windows Media Foundation H.264 编码链路；
- Android MediaCodec 硬件 H.264 解码和低延迟参数；
- 1920 × 1080、目标 50 FPS、16 Mbps 视频模式；
- 最新帧优先的有界捕获管线和延迟统计；
- 集成 Windows 主机、驱动、ADB 和 Android APK 的版本化安装包。

### 修复

- 改进扩展屏系统鼠标显示；
- 修复 USB 重连与 Windows 控制端启动异常；
- 降低高动态画面播放时的卡顿和延迟积累。

## 0.2.0 - 2026-07-10

### 新增

- 首个带统一版本号的 Windows x64 一键安装包；
- 安装包集成 Windows 主机、间接显示驱动、软件设备启动器、ADB 和 Android APK；
- 预留更新配置、运行时停止脚本和完整卸载接口；
- Windows 控制中心支持启动、停止、重连和状态刷新。

### 安全

- 开发版驱动使用测试签名，并在安装时明确提示测试签名环境要求；
- 安装程序不会自动修改 BCD、安全启动或重启配置。
