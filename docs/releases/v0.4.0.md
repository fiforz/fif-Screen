# FifScreen 0.4.0

本版本为 Android 扩展屏加入 Windows 原生多点触控。

## 更新内容

- 在手机或平板画面上直接点击、拖动和长按 Windows 扩展桌面；
- 支持最多 16 个触点，双指缩放和滚动由目标 Windows 应用原生处理；
- 触控不会强制移动系统鼠标，鼠标和触控可以独立使用；
- Android 使用独立低延迟发送线程，并合并连续移动帧，避免输入积压；
- Windows 将归一化触点坐标映射到 FifScreen 扩展显示器并调用系统触控注入；
- 新旧组件通过握手协商触控能力，旧版本仍可继续显示画面；
- USB 或控制连接断开时自动取消活动触点，避免残留按下状态；
- 增加协议、坐标映射、单指、多指和断线取消自动测试。

## 安装说明

这是开发预览版，包含测试签名显示驱动。Windows 必须已允许测试签名驱动，安装程序不会修改 BCD 或安全启动设置。
