# FifScreen 0.4.1

本版本修复 Android 画面可以显示、但触摸 Windows 扩展桌面无响应的问题。

## 更新内容

- 控制中心会校验 Windows 主机和设备启动器版本，避免误启动旧构建产物；
- 发现旧主机时只替换主机进程，保留已存在的扩展显示设备和 Owner 进程；
- Android 端始终消费完整触摸手势，避免连接状态切换时丢失移动或抬起；
- 增加触摸能力协商、输入就绪和发送结果日志，便于定位不同 Android 设备的问题；
- 已在真实 Android 设备和 1920 x 1080 FifScreen 扩展屏上验证点击可产生 Windows 原生 `WM_POINTERDOWN/UP`，并兼容普通鼠标点击消息。

## 安装说明

这是开发预览版，包含测试签名显示驱动。Windows 必须已允许测试签名驱动，安装程序不会修改 BCD 或安全启动设置。
