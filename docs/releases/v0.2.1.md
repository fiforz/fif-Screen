# FifScreen 0.2.1

本版本完成了可实际使用的 USB 扩展屏低延迟链路。

## 更新内容

- 新增 Windows Media Foundation H.264 硬件编码；
- 新增 Android MediaCodec H.264 硬件解码；
- 默认提升至 1920 × 1080、目标 50 FPS、16 Mbps；
- 使用最新帧优先的有界捕获管线，避免延迟持续累积；
- 改进扩展屏鼠标显示与 USB 断线重连；
- 提供集成驱动、ADB、Android APK 和控制中心的 Windows 安装包。

## 安装说明

这是开发预览版，包含测试签名显示驱动。Windows 必须已允许测试签名驱动，安装程序不会修改 BCD 或安全启动设置。
