# FifScreen 0.2.0

这是 FifScreen 首个提供统一版本号和 Windows 一键安装包的开发预览版。

## 更新内容

- 集成 Windows 主机、间接显示驱动和软件设备启动器；
- 集成 Android Platform Tools ADB 运行环境；
- 把 Android APK 封装到 Windows 安装包内；
- 提供 Windows 控制中心，可启动、停止、重连和刷新状态；
- 预留版本更新配置、运行时关闭和完整卸载接口；
- 所有组件统一使用 `0.2.0` 版本号。

## 安装说明

这是开发预览版，包含测试签名显示驱动。Windows 必须已允许测试签名驱动，安装程序不会修改 BCD 或安全启动设置。
