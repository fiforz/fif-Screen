# FifScreen 0.5.0

本版本在保留现有 USB 调试连接的基础上，新增无线局域网扩展屏。

## 更新内容

- Windows 控制中心与 Android 应用均可手动选择“USB 调试”或“无线局域网”；
- 两台设备连接同一局域网并输入相同四位 PIN 后，Android 会自动发现电脑并连接；
- 无线模式支持与 USB 一致的 1920 x 1080、目标 50 FPS H.264 画面和 Windows 原生触控；
- PIN 使用 PBKDF2-HMAC-SHA256 挑战响应完成客户端证明、主机证明和独立视频通道认证；
- 无线端口的 Windows 防火墙规则仅允许本地子网访问，卸载 FifScreen 时会自动移除；
- 代码已预留服务器端点目录和中继传输接口，当前版本不连接任何服务器；
- USB 模式仍只监听 `127.0.0.1` 并使用原有 ADB reverse，不需要输入 PIN。

## 使用无线连接

1. 首次使用先通过 USB 让 Windows 控制中心安装 Android APK；
2. 确保电脑和 Android 设备位于同一个可信局域网；
3. Windows 控制中心选择“无线局域网”，输入四位 PIN 并启动扩展屏；
4. Android 应用按返回键打开连接设置，选择“无线局域网连接”并输入相同 PIN。

无线协议当前提供握手认证，但画面和触控流尚未端到端加密，请勿在公共或不可信网络使用。

## 安装说明

这是开发预览版，包含测试签名显示驱动。Windows 必须已允许测试签名驱动，安装程序不会修改 BCD、安全启动设置或主动重启电脑。
