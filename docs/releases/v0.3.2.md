# FifScreen 0.3.2

本版本修复 Windows PowerShell 5.1 读取 GitHub Release 静态更新清单时的兼容性问题。

## 更新内容

- GitHub Release API 暂时不可用时，继续回退到最新 Release 的静态更新清单；
- 对 `application/octet-stream` 返回的清单内容显式执行 UTF-8 解码，不再把字节数组误当成 JSON 对象；
- 静态清单解析完成后继续校验产品、更新通道、架构、版本、安装包地址和 SHA-256；
- 更新检查显式启用 TLS 1.2，并关闭 PowerShell 下载进度渲染，降低请求阻塞概率；
- 增加二进制静态清单解析的自动回归测试。

## 安装说明

这是开发预览版，包含测试签名显示驱动。Windows 必须已允许测试签名驱动，安装程序不会修改 BCD 或安全启动设置。
