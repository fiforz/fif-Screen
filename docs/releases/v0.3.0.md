# FifScreen 0.3.0

本版本补齐中文桌面控制端和基于 GitHub Releases 的完整自动更新链路。

## 更新内容

- 控制中心启动后自动检查 GitHub 最新 Release；
- 新版本提示包含版本号、发布时间和更新日志；
- 自动下载安装包并执行 SHA-256 完整性校验；
- 安装前关闭 FifScreen 主机、设备启动器、控制中心和安装目录内 ADB；
- 网络不佳时在后台静默退避重试；
- 手动检查失败时提供明确中文网络提示；
- Windows 控制中心按钮、状态和主要日志全面中文化；
- 新增“关于 → 检查更新 / 使用教程 / 关于 FifScreen”；
- 新增 Android 开发者模式、USB 调试、连接和故障排查教程；
- 重写中文 README，并加入标准化版本发布脚本。

## 安装说明

这是开发预览版，包含测试签名显示驱动。Windows 必须已允许测试签名驱动，安装程序不会修改 BCD 或安全启动设置。
